Definitions

Ao3 Workskin —
Ao3 allows you to add CSS to your work with 'workskins'. Like the example above, you need the workskin to add more than just basics. One of the more popular things to do with workskins is to use it to mimic different things - like emails, messages, and social media sites. it can add a lot of immersion and make your fic look a lot cooler, especially if its a chat fic or socmed fic.

Let's take a look at an example with the 'simple messaging' workskin:

HTML —
aka "HyperText Markup Language", code used to create webpages. You put text inside tags that makes it behave a certain way, like adding a paragraph, making the text bold, or creating a list.

HTML

<p> I can be <b>bold</b>, or even <i>italics!</i> </p>

TEXT

I can be bold, or even italics!


CSS —
aka "Cascading Style Sheets", code that adds style and structure to a webpage. It's usually put in a separate file, and then you use HTML tags to add it to the text.

CSS

.pinkbox{ background-color: pink;
padding: 5px;}

HTML

<p> Wow, there's a <span class = "pinkbox">pink box! </span> </p>

WITHOUT CSS

Wow, there's a pink box!

TEXT WITH CSS

Wow, there's a pink box!

How do I use the Workskin Workshop?

  1. Go to the generator you want to use

    2. I've added examples so you can see what everything looks like. Make sure to read the Notes so you know how it works

  2. Generate the HTML and add it to your fic

    3. Click the 'Generate HTML' button, then click 'Copy'. Go to edit your fic, place the HTML where you want it. Remember, it won't look like it does in the generator until you add the workskin

  3. Generate the CSS and create a workskin

    4. Go to the CSS generator and generate what you will use, then add it to a workskin. There is a more indepth tutorial on creating workskins here. After that, go to your fic and add the workskin to it.

  4. Check everything works

    5. Check that what you added looks good on both desktop and mobile. All good? If not, check if you copy-pasted the HTML or CSS wrong. If it still doesn't work, or if you notice something off, send me a message.

How do I create my own workskin?

I'm so glad you're interested! Although I try and add a lot of options when making the generators, it will never be as customizable as manipulating it yourself, or creating your own!

Firstly, you need to know the basics of HTML and CSS. Next, you need somewhere to edit your code. I wouldn't write it directly on Ao3, you WRITE. I, personally, use VSCode to write and edit the HTML/CSS. I understand that it can be intimidating to download a whole ass program for making workskins, but with VSCode, you can download Microsoft Live Preview to see how each change you make affects your code in real time! I've also added a link that will take you to an HTML specific guide for using VSCode. It's for making your own website, but the setup is the exact same.

I am planning on making a workskin walkthrough at some point, so look out for that! But at the moment, if you have any problems (even if you can't get VSCode to work for whatever reason), please message me with your discord and I'll help with what I can.

For now, some general advice:
  • The CSS Ao3 allows you to use is limited. I've been told this might change in the future, but for now a lot of more modern HTML won't work
  • You can use only the default of flex
  • Ao3 has some defaults for paragraphs and divs that will make your HTML look different than it does on VSCode / normally. Right click on whatever looks wonky, then click inspect and then look at the CSS section, it will tell you what's affecting what element.
  • Ao3 automatically adds paragraphs after divs, and a few other situations, so if your code looks wonky it might also be that.
